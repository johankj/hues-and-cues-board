# Hues and Cues Board

A web-based recreation of the Hues and Cues board game color grid with random selection feature.

**Live Demo:** https://hues-and-cues-board.netlify.app/

## Features

- 16Ã—30 color grid (rows A-P, columns 1-30) with exact colors from the original board
- Grayscale reference strip
- Random color selection
- Manual cell selection by clicking/tapping
- Color preview panel (toggleable)
- Responsive design for desktop, tablet, and mobile

## Usage

- **Random Select** - Pick a random color from the board
- **Click/Tap** - Select any color cell manually
- **Clear Selection** - Deselect the current color
- **Hide/Show Preview** - Toggle the color preview panel

## Development

Static HTML/CSS/JS - no build process required. Open `index.html` in a browser or deploy to any static hosting service.
